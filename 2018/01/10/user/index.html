用户

发送验证码


测试时默认验证码都是1234
  • 访问方式
    POST
  • 访问地址
    /api/user/code

  • 访问参数

    参数 | 含义 | 类型 | 说明
    :—-: | :—-: | :—-: | :—-:
    mobile | 手机号码 | int |
    check_register | 检查注册与否 | bool | 1是 / 0否 (比如忘记密码这个接口使用就不需要检查注册与否)

  • 返回属性

    参数 | 含义 | 类型 | 说明
    :—-: | :—-: | :—-: | :—-:
    result | 发送验证码结果 | bool | true 成功 / false 失败

校验验证码

  • 访问方式
    POST
  • 访问地址
    /api/user/code/check

  • 访问参数

    参数 | 含义 | 类型 | 说明
    :—-: | :—-: | :—-: | :—-:
    mobile | 手机号码 | int |
    code | 验证码 | int |

  • 返回属性

    参数 | 含义 | 类型 | 说明
    :—-: | :—-: | :—-: | :—-:
    result | 校验结果 | bool |
    voucher | 验证码使用凭证 | string | 后续操作用此凭证

注册(设置密码)

  • 访问方式
    POST
  • 访问地址
    /api/user/register

  • 访问参数

    参数 | 含义 | 类型 | 说明
    :—-: | :—-: | :—-: | :—-:
    password | 密码 | string |
    password_confirmation | 确认密码 | string |
    mobile | 手机号码 | int |
    voucher | 验证码使用凭证 | string | 上一步校验验证码获取的凭证

  • 返回属性

    参数 | 含义 | 类型 | 说明
    :—-: | :—-: | :—-: | :—-:
    name | 用户名 | string |
    avatar | 头像 | string|
    meta | 登陆状态保存 | array |
    meta.token | 登陆凭证 | string | 参考基础文档说明
    meta.expire_in | 过期时间 | |
    meta.refresh_in | 刷新过期时间 | | 此参数对对住我选项有用

    meta.expire_in | 过期时间 | |
    meta.refresh_in | 刷新过期时间 | | 此参数对对住我选项有用
